# CS115.Q13-Singular-Value-Decomposition-and-Principal-Component-Analysis
This repository contains demo implementations and presentation materials for the course project  
**Singular Value Decomposition (SVD)** and **Principal Component Analysis (PCA)**.

The project aims to provide a clear theoretical overview and practical demonstrations of PCA and SVD, including dimensionality reduction and image compression applications.
